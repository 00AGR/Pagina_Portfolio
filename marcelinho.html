Marcelinho Cata Ovos

Jogo: Marcelinho Cata Ovos
Autores: Agratha Cristina, Amanda Machado, Gabriela Ambone
Estilo: 2D Pixel Art
Gênero: EndlessRunner
Desenvolvimento Web
Programação: HTML, JavaScript

Marcelinho Cata Ovos foi desenvolvido como um jogo em página web do componente de Desenvolvimento Web do curso de Programação de Jogos Digitais do IFPR. No jogo, Marcelinho deve fugir das galinhas que correm furiosas, enquanto tenta desviar de obstáculos, após ter entrado no galinheiro para pegar ovos e ter chamado a atenção das galinhas do recinto. O jogo é baseado em uma história real de uma das integrantes da equipe desenvolvedora.
