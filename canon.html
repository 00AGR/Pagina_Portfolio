Cannon Wars

Jogo: Cannon Wars
Autores: Agratha Cristina
Estilo: 2D ASCII Art
Gênero: Casual
IDE: CodeBlocks
Programação: C++

Cannon Wars foi desenvolvido para o componente de Programação durante o primeiro ano do curso de Programação de Jogos Digitais do IFPR. É um jogo competitivo de canhões para dois jogadores. O ângulo do tiro pode ser ajustado através das setas do teclado, e a força é definida pelo gauge no momento em que o jogador pressionar para atirar. O trajeto do projétil é calculado e caso acerte o canhão inimigo, o jogador marca um ponto, até um dos dois jogadores vencer a partida.
