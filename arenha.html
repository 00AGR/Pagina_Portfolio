Dona Arenha

Jogo: Dona Arenha
Autores: Agratha Cristina, Amanda Machado
Estilo: 2D Arte Vetorial
Gênero: Casual
Motor Gráfico - Unity Engine
Programação: C#

Dona Arenha foi desenvolvido como um projeto de jogo mobile para o componente curricular de Desenvolvimento Para Dispositivos Móveis do curso de Programação de Jogos Digitais do IFPR. A ideia do jogo consiste em dois amigos que tentam assoprar uma aranha pendurada em sua teia, para que caia em um dos dois. É um jogo competitivo para jogar com um amigo, quem assoprar até derrubar a aranha no seu adversário, ganha.
